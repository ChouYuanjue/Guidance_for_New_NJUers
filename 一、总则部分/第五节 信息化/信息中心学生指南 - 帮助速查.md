## <font style="color:rgb(0, 0, 0) !important;">一、统一身份认证</font>
在南京大学，学号是大家的通行证。所有的校内服务都会依托于学号进行身份验证。**<font style="color:#4C16B1;">统一身份认证</font>**就是这样的一套系统。<u>（你将会在后续的教程中多次遇到这个词，请记住他）</u>目前，已经有<font style="color:rgb(51, 51, 51);">微信公众号“南京大学信息门户”、南京大学APP、上网认证系统</font>`<font style="color:rgb(51, 51, 51);">p.nju.edu.cn</font>`<font style="color:rgb(51, 51, 51);">、网上办事大厅</font>`<font style="color:rgb(51, 51, 51);">ehall.nju.edu.cn</font>`<font style="color:rgb(51, 51, 51);">、支付通、就业系统、学生第二课堂、VPN服务（Easy Connect） 等多项服务接入了统一认证系统。</font>

### <font style="color:rgb(0, 0, 0);">常见问题</font>
1. **<font style="color:rgba(0, 0, 0, 0.85) !important;">如何登录统一认证？</font>**<font style="color:rgba(0, 0, 0, 0.85) !important;">  
</font><font style="color:rgb(0, 0, 0);">输入同一个账号密码（统一身份认证的账号名、密码）即可登录进入。</font>

<font style="color:rgb(0, 0, 0);">电脑端可使用</font><font style="color:#4C16B1;">已认证并登录</font><font style="color:rgb(0, 0, 0);">的手机微信或南京大学APP扫码登录。</font>

_<font style="color:rgba(0, 0, 0, 0.85);">具体步骤见 “统一身份认证” 第 1 问。</font>_

<u><font style="color:rgba(0, 0, 0, 0.85);"></font></u>

2. **<font style="color:rgba(0, 0, 0, 0.85) !important;">不清楚统一身份认证的账号名和初始密码？</font>**<font style="color:rgba(0, 0, 0, 0.85) !important;">  
</font><font style="color:rgb(51, 51, 51);">统一身份认证的用户名为</font><font style="color:#4C16B1;">学号</font><font style="color:rgba(0, 0, 0, 0.85);">；初始密码为</font><font style="color:#4C16B1;">全部身份证件号</font><font style="color:rgba(0, 0, 0, 0.85);">。</font>

首次使用统一身份认证时，需要根据网页提示<font style="color:#4C16B1;">更改密码</font>。

_<font style="color:rgba(0, 0, 0, 0.85);">详见 “统一身份认证” 第 2 问。</font>_

<u><font style="color:rgba(0, 0, 0, 0.85);"></font></u>

3. **<font style="color:rgba(0, 0, 0, 0.85) !important;">登录时提示 “用户名或密码错误”“账号冻结”“密码过期”？</font>**<font style="color:rgba(0, 0, 0, 0.85) !important;">  
</font><font style="color:rgba(0, 0, 0, 0.85);">密码错误可通过 “忘记密码”（需绑定手机 / 邮箱）或邮件申请重置；</font>

<font style="color:rgba(0, 0, 0, 0.85);">账号冻结半小时后自动恢复；</font>

<font style="color:rgba(0, 0, 0, 0.85);">密码过期需重置密码。</font>

_<font style="color:rgba(0, 0, 0, 0.85);">详见 “统一身份认证” 第 3、4、10 问。</font>_

<u><font style="color:rgba(0, 0, 0, 0.85);"></font></u>

4. **<font style="color:rgba(0, 0, 0, 0.85) !important;">密码登录时需要短信验证，但收不到验证码？</font>**<font style="color:rgba(0, 0, 0, 0.85) !important;">  
</font><font style="color:rgba(0, 0, 0, 0.85);">需在 “统一身份认证平台”（</font>[<font style="color:rgb(9, 105, 218);">https://authserver.nju.edu.cn</font>](https://authserver.nju.edu.cn/)<font style="color:rgba(0, 0, 0, 0.85);">）绑定手机号，可通过微信信息门户 “个人中心”、南大 APP “认证账号设置” 或电脑端平台操作</font>

_<font style="color:rgba(0, 0, 0, 0.85);">详见 “统一身份认证” 第 5、6 问。</font>_

<u><font style="color:rgba(0, 0, 0, 0.85);"></font></u>

5. **<font style="color:rgba(0, 0, 0, 0.85) !important;">更换学号后，如何重新登录微信信息门户、南大 APP 或绑定新手机号？</font>**<font style="color:rgba(0, 0, 0, 0.85) !important;">  
</font><font style="color:rgba(0, 0, 0, 0.85);">需先解绑旧学号，再用新学号登录，解绑步骤及新登录方式</font>

_<font style="color:rgba(0, 0, 0, 0.85);">见 “统一身份认证” 第 7、8、9 问。</font>_
