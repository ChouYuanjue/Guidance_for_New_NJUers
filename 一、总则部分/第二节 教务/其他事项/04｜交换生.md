## 交换
交换是指，在南京大学就读期间，前往其他高校修读相关课程或者进行相关研究。交换期间，学生的学籍仍然保留在南京大学，一般来说，也无法取得交换学校的相应学位。**交换生只能在本校获取学位**。交换项目根据时间长短可以分为短期和长期两种。交流时间少于3个月的项目为短期项目；3个月以上的项目则为长期项目，一般是一学期或者一学年。

### 为什么参加交换项目
主要有几个原因：感受外国文化、理念与生活；提高外语能力；增加和丰富学习与科研经历；拓展知识面；学习国外先进技术。其中最重要的是第一个原因，由于学生出国交换时限一般为3-12个月，基本在这段时间内仅仅足以被新国家的人文适应、语言适应所消耗。学生出国交流更重要的是感受外国的先进文化和生活理念，也许这些感受不能在短期内给学生带来立竿见影的效果，但也许在许多年以后，会对学生今后人生的选择或事业与生活产生潜移默化的影响。



目前，南京大学的国内交换项目主要是与**C9联盟高校之间**的交换，在每年的4-6月份进行报名，近两年年来的交换主要是北京大学、西安交通大学、上海交通大学、哈尔滨工业大学。

国外交换有主要有两种方式，一种是通过[南大国际交流处](https://stuex.nju.edu.cn/)报名的交换，这种交换是由学校或学院牵头的，通常是和南京大学签有交换协议的学校，一种是自己套磁国外导师进行的暑研等

### 国际交流处
南京大学是国际交流最为活跃的中国大学之一，目前南京大学已经与世界30多个国家的280多所高校签署了校级学生交流协议，这些高校遍布欧洲、亚洲、南北美洲和大洋洲。通过国际交流处的项目主要有5类

+ 免学费项目：由与我校签署过学生交流协议的伙伴学校发布的项目，录取人数有限，对于录取学生，仅免除学费，其他费用（食宿、机票、生活费）自理
+ 自费项目：由于我校签署过学生交流协议的伙伴学校发布的项目，录取人数根据伙伴学校要求选定，项目执行一般在寒暑假，所有费用需要自理
+ 国家留学基金委项目：就是通常所说的[CSC](https://www.csc.edu.cn/)项目，在出国板块我们会再次看到它，这里主要讨论的是交换项目
    - 优秀本科生国际交流项目：选派类别为本科插班生，和大部分同学无关，这里不再赘述
    - 国家公派硕士研究生项目：面向高等学校本科毕业生选拔攻读硕士学位研究生，或面向高等学校在读硕士生选拔联合培养研究生，攻读期限一般为12-24个月，联合培养的留学期限在3-12个月。国家留学激进自主一次往返国际旅费和规定期间的奖学金，资助标准按照国家相关规定执行
    - 需要注意：国家留学基金委通常会有回国服务期的协议，即学成回国后需要在国内工作一定年限，但由于很多学生由于各种原因违约，近年来CSC项目的审核在不断收紧，目前主要是以研究型博士学位的资助为主
+ 欧盟项目：项目申请和评审期较长，分为Experts, Lotus, Erasmus等系列，免学费，且有一定生活补贴
+ 学校提供的奖学金项目：
    - 徐新国际交流奖学金：每年选派12名本科生，提供5w美金（约合人民币35万），赴牛津大学、剑桥大学、普林斯顿大学、哥伦比亚大学、斯坦福大学、麻省理工学院、康奈尔大学、宾夕法尼亚大学、布朗大学和达特茅斯学院交换学习一年。该项目需要进行选拔.
    - 学工处奖学金项目：每年选派机动名额的贫困本科生，资助3-5万人民币赴牛津大学、英国约克大学、美国代顿大学进行短期暑期交换

### 暑研
在进行国外院校的申请时，尤其是北美的研究型硕士项目，暑研是一个非常好的提升软背景的机会，学校可以为本科生提供一定的暑研机会，但也可以自行套磁相关院校的教授。暑研完成后，有很大概率可以得到一篇高质量的论文产出和海外教授写的推荐信。有的导师甚至会用自己的经费招募暑研学生。一些境外的院校也有专门的暑研计划，如UCLA的[CSST项目](https://csst.ucla.edu/)，以及港校在大三暑期开展的提前批次的招生和workshop



## 交换的要求
一般来说，交换的要求比相应的国外大学申请要低。一般来说需要满足几个基本条件

+ 学分绩4.0/5.0，也就是均分在80以上，无不及格科目
+ 语言成绩：很多交换院校可以接受CET-6的成绩，所以还没有考出托福雅思的同学也不用担心。交换项目的语言成绩一般要求不高，IELTS 6-6.5, TOEFL80左右，但如果之后有出国留学的需求的话，还是要比这个要高一点的
+ 有些学校会有自己的一套标准（如UCB），需要参加对方的面试
+ 一般来说，自费项目的要求较低

## 交换学分认定
1. 学分认定范围：**只有学校和学院的官方项目可以转换学分，自己找的项目不可以**
2. 课程认定基本原则
+ 对方院校或对应学科的学术声誉、学术地位与我校相当
+ 凡是对方学校所修课程内容与学生所在院系专业指导性教学计划无法对应的，统一认定为选修课程，符合我校通识教育课程建设理念及标准的可以认定为通识课程
+ 课程学分与学时对应关系：1学分=16学时
+ 交换成绩以百分制记载，如不是百分制，则折算成百分制
+ **校内已经修读的课程，不得用校外交流课程替代**
3. 具体认定方法
+ 学生参与交换的学期，须于开学两周内登录本科生教服平台完成交流项目备案
+ 学生在完成交流以后，向所在院系提供境内外学校学习的成绩单原件
+ 依据成绩单，学生登陆教服平台填写《交流课程学分认定申请表》
+ 将填写好的申请表附境内外学校提供的成绩单原件和课程介绍、 学时证明等材料， 报相关院系审核课程性质、学分和成绩， 经相关院系审核后由教学院长或主任签字并加盖院系公章 
+ 教务处对学生提交的《交流课程学分认定申请表》及相关材料进行复审并完成成绩录入 
+ 课程认定结束后，《交流课程学分认定申请表》留存教务处，境内外学校提供的成绩单原件留存学生所在院系，最后并入学生学籍档案。
