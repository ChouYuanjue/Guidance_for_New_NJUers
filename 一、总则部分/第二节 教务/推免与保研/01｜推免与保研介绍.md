保研，推免都是免初试保送研究生的术语，下面向大家介绍具体的概念。

**<u>保研分为两个步骤：拿到南大给你的推免名额；得到南大或者其他学校的保研offer。</u>**

广义上的保研包括：**<u>普通保研、支教保研、军队院校补偿计划、强基转段、港澳台申请。</u>**本文只介绍普通保研，其余各种形式的保研信息需要自行查找。



首先是第一步，拿到南大给你的推免名额。在大四第一学期开学，也就是9月初的时候。每个学院将会公布本学院拿到推免名额的学生名单。推免名额由学校协调分配给每个学院，因此每个学院的“推免率”一般都是不一样的。

学院如何分配推免名额呢？在南京大学，推免名额基本都是你自己的学院（或者是专业）根据成绩来进行分配的（保研课程的成绩，具体可见前文），保研课程清单也是每年9月公布（一般来说每年变化不会太大）。**<u>成绩可以说占到了90%以上的权重</u>**，有一些竞赛或者科研成果是可以增加推免的成绩计算的，但是比较困难，每年能加分的人也在少数。

虽然说最终名单9月才公布，但是在第五学期结束，或者第六学期结束放暑假的时候，大部分人都可以根据交换生系统的排名或者自己心里大概对自己能否拿到推免有数了。

如果前面的人放弃了推免名额（可能是出国或者本科就业或者出国），那么名额将会继续顺延到70%（也就是说，如果你排名70%之后，即使此时名额有剩余，那也无法保研）。同时，港澳台同学由于政策原因，无法拿到保研名额。



第二步，拿到各大学校的保研offer。

在大三下的5月份到大四上的9月份，各大高校会组织各种形式的保研考核（例如：夏令营、预推免等等，本质上都是考保研考核）。各大学校的学生可以将自己的**<u>本科前几年的成绩与排名、学校专业背景、英语水平、科研竞赛能力</u>**写在自己的简历上，提交到希望保研的各大高校通过初审，然后去参加校测（笔面试等）获得保研offer。

由于同一个学生可以参加多个学校的保研考核，但最终只能选择一个offer，因此有些学校会超发一定数量的offer/优秀营员，所以对一些学校来说，拿到保研考核的offer/优秀营员并不一定就代表最终可以保研去这个学校（但是也有很多学校是铁offer，即发offer=要你），所以往往一个同学会去参与很多学校的保研考核，多拿一些offer。最终在9月28日，推免系统开放，你可以选择最终确定的1个offer作为你的最终保研去向。

至于保研**<u>本校本院</u>**，南大各院都或多或少有一些本校生的保护主义政策，有时是潜在的面试加分、有时是不公开的名额划分，总之按照学院的规定来就好，不需要额外注意什么。但是，按现行政策及今年的保研情况，保本校也依然会有筛选，即**<u>保本校本院不成功的情况常有发生（通常见于成绩擦线保研的同学）</u>**，所以建议同学们最好还是多拿几个学校的offer保底。

如果希望**<u>跨专业保研</u>**的同学也是同理。例如：本科是数学专业的同学，如果希望保研到计算机专业。则需要将自己的简历提交到某校计算机学院的保研考核项目（往往需要本科多做一些对应的内容以充实简历），通过**<u>初审、校测</u>**，最终拿到计算机专业的offer。



强基计划、港澳台、医学院长学制、本硕博贯通（至诚班）等等特殊专业的免试研究生政策根据当年具体情况和政策施行

**<font style="color:rgb(128, 98, 194);">夏令营</font>**

<font style="color:rgba(0, 0, 0, 0.9);">在9月之前，最常见的可以获得研究生offer的方式叫</font>**<font style="color:rgb(250, 83, 89);">夏令营</font>**<font style="color:rgba(0, 0, 0, 0.9);">。学生申请报名之后，接收方会进行初审，审核通过的学生称为</font>**<font style="color:rgb(250, 83, 89);">入营。</font>**

**<u>入营的学生将被接收方进行筛选，一般而言有两种筛选模式，</u>****<font style="color:rgb(250, 83, 89);">强com和弱com</font>**

<font style="color:rgba(0, 0, 0, 0.9);">这里的com，指的是committee，即</font>**<font style="color:rgb(47, 118, 195);">接收方的招生委员会</font>**<font style="color:rgba(0, 0, 0, 0.9);">。和com对立的，是接收方学院的</font>**<font style="color:rgb(47, 118, 195);">导师</font>**<font style="color:rgba(0, 0, 0, 0.9);">。</font>

**<font style="color:rgb(47, 118, 195);">强com，即委员会的招生话语权更大</font>**<font style="color:rgba(0, 0, 0, 0.9);">，一般的筛选形式是统一进行考核，可能包括笔试，面试等。</font>

**<font style="color:rgb(47, 118, 195);">弱com，即导师的话语权更大</font>**<font style="color:rgba(0, 0, 0, 0.9);">，很多时候由导师决定是否收学生。</font>

<font style="color:rgba(0, 0, 0, 0.9);">当然，强com和弱com是相对的，且没有明显界限，</font>**<font style="color:rgb(47, 118, 195);">对于很多项目来说，你强就是弱com，你弱就是强com</font>**<font style="color:rgba(0, 0, 0, 0.9);">。</font>

<font style="color:rgba(0, 0, 0, 0.9);">统一筛选后，</font>**<font style="color:rgb(250, 83, 89);">会有一部分学生获得offer，另一部分学生获得候补（waiting)资格</font>**<font style="color:rgba(0, 0, 0, 0.9);">。接收方会先询问获得offer的学生是否接下offer，如果有放弃的情况，则offer会顺延给候补名单中的学生。</font>

**<font style="color:rgb(250, 83, 89);">会有一些学院不开设夏令营</font>**<font style="color:rgba(0, 0, 0, 0.9);">，想要推免过去，需要直接联系导师，</font>**<font style="color:rgb(47, 118, 195);">这种模式可以理解为另一种弱com夏令营</font>**<font style="color:rgba(0, 0, 0, 0.9);">。  
</font>

**<font style="color:rgb(128, 98, 194);">预推免、九月（十月）推免</font>**

<font style="color:rgba(0, 0, 0, 0.9);">如果接收方夏令营在8月以后还有空缺名额，会开放</font>**<font style="color:rgb(250, 83, 89);">预推免</font>**<font style="color:rgba(0, 0, 0, 0.9);">。预推免作为</font>**<font style="color:rgb(47, 118, 195);">补录</font>**<font style="color:rgba(0, 0, 0, 0.9);">机会，流程会比夏令营更简单，统一简单考核居多。</font>

**<font style="color:rgb(47, 118, 195);">不开设夏令营的院系，主要的推免方式就是预推免</font>**<font style="color:rgba(0, 0, 0, 0.9);">。</font>

**<font style="color:rgb(47, 118, 195);">九月（十月）推免</font>**<font style="color:rgba(0, 0, 0, 0.9);">则是在推免的最后阶段，接收方的最后一次补录，通常名额很少，且信息差较大。</font>

**<font style="color:rgb(128, 98, 194);">学生能否拿到心仪的研究生offer，主要凭借的就是个人背景和在这些推免机会中的努力了</font>**

**<font style="color:rgb(128, 98, 194);">保研去向：学硕、专硕和直博</font>**

<font style="color:rgba(0, 0, 0, 0.9);">一般而言，保研可能被录取为</font>**<font style="color:rgb(250, 83, 89);">学硕、专硕、直博</font>**<font style="color:rgba(0, 0, 0, 0.9);">三种学制。</font>

<font style="color:rgba(0, 0, 0, 0.9);">学硕，即学术型学位研究生，以学术研究为导向，偏重理论和研究。</font>**<u></u>**

<font style="color:rgba(0, 0, 0, 0.9);">专硕，即专业学位硕士研究生，是与特定职业领域相关的硕士学位。</font>

**<u>直博，则是直接攻读博士学位。</u>**

**<u></u>**

**<font style="color:rgb(250, 83, 89);">特殊的：</font>****<u></u>**

1.如果选择读学硕，一般可以选择在研二<font style="color:rgb(47, 118, 195);">转博</font>（一般需要通过博资考）。

2.学硕和博士都是学校中的科研人员，<font style="color:rgb(47, 118, 195);">一般在课题组会有工资，且学硕的学费一般比较低（普遍为8000/年）</font>。而专硕并不严格属于科研人员，还享受了学校的很多课程、就业资源，<font style="color:rgb(47, 118, 195);">一般不仅没有工资，还需要交更多学费</font>（高低不一，一年一两万-20多万都存在）

3.对于大部分文理工科而言，<font style="color:rgb(47, 118, 195);">由于学硕能进能退，一般录取难度高于专硕和直博</font>。对于医学而言<font style="color:rgb(47, 118, 195);">一般直博最难（博士是就业门槛）</font>，对于商科而言，<font style="color:rgb(47, 118, 195);">专硕一般最难（就业结果最好</font>）
