出国留学是很多南大同学在进行完本科、硕士或者博士阶段的学习以后一种升学方式，其优点在于，可以近距离接触西方的社会，在国外往往也能学到一些前沿的科研领域（现阶段，以美国为代表的西方国家还是在很多尖端领域掌握话语权），缺点在于，需要良好的英语能力；西方国家的主流价值观与我们不同，会有一段适应期；长期不能回家；花费较高；随着中国的不断崛起，海归的优势在不断降低；可能会受到歧视等。

一般来说，出国（境）的主要地区按照体系分为北美（美国、加拿大），英联邦（英国、澳洲，新加坡，中国香港），欧陆（瑞士、德国、法国）等。为了叙述方便，本文中的出国就是指代出国（境）。

## 硕士项目
一般来说，国外的硕士项目主要分为授课型和研究型两大类，此外还有LLM，MBA等专业硕士项目。这里主要讨论授课型和研究型硕士。授课型和研究型有点类似于国内的专硕和学硕，但两者的差别更大。

### 授课型硕士
授课型硕士，顾名思义，就是以上课考试和项目为主的硕士，一般年限较短，1-2年即可完成，没有指定的导师，没有论文要求，大部分无需撰写毕业论文。北美的Meng项目，英联邦的Msc硕士项目基本都是授课型，欧陆的硕士大多是两者的结合，即以授课为主，但也要撰写毕业论文，进行一些实验室研究。授课型一般被称为“水硕”，但我们也要辩证的看待它的存在。授课型一般有如下的优势和劣势

+ 授课型硕士年限较短，尤其是针对商科这种需要大量实习工作经验的学科，或者cs这种技术为王的学科，如果在国内的组没有碰上很好的导师，或者导师不放实习的话，找工作的难度将大大增加；一年制硕士回内地具有较大的年龄优势，如果本科能力已经很强或者有很明确的就业需求的话，去快速拿学历也是一种思路
+ 授课型硕士主要面向就业，对以后申博没有实质性帮助，所以不适合有学术追求的同学；像商科cs等学科出国以后可能有较大的机会留在国外工作积累经验，但一些基础学科的难度留在当地的难度则会大大增加
+ 申请相对简单，尤其是港新的授课型硕士，一直受到国内不想考研同学的青睐
+ 授课型硕士为全自费项目，价格较为昂贵
+ 与此同时，因为其年限短，所以对本科的背景和能力极其看重，一年的时间（实际只有8-10个月左右）要克服文化语言的隔阂，完成任务不轻的课程，而且从一入学开始就要开始找工作，是非常赶的
+ 关于含金量：英联邦的一年制硕士方差极大，但含金量极大的倚重本科，而南大作为华五高校，在国内享有极大的声誉，一般来说南大本硕的简历和南大本授课硕的简历都不会被卡，但光光过简历关是远远不够的，这意味着，想找到好的工作还必须提升自己的个人能力

### 研究型硕士
研究型硕士（Mphil），就是以研究为主要形式的项目，上的课很少，主要精力是和导师一起完成实验室的任务，年限一般为1.5-2年。研究型硕士的待遇一般和博士一样，有很大的概率得到奖学金。英联邦的研究型硕士很少，且一般不招收中国大陆的学生，所以这里主要讨论北美和欧陆的研究型硕士的优势和劣势

+ Mphill有奖学金，开销少
+ 欧陆大部分国家不可以本科直博，所以想在欧陆读博，一般要先在欧陆读硕士，刚刚提到欧陆的硕士属于授课为主，但也需要做一定研究，撰写毕业论文
+ 研究型硕士不是以就业为导向的，所以可以理解为博士的预科。尽管如此，北美和欧陆的研究型硕士毕业出来就业的前景依然可观
+ 可以真正和导师进行实验室科研，学习国外的先进技术，比国内学硕时间安排更加紧凑
+ 申请难度比较大，有些项目需要提前联系导师。一般需要本科时有较好的学术表现，最好有paper发表（ETH这种分数控除外）



关于研究型和授课型，总的来说是要根据自己的未来规划和当前的需求而决定的，项目本身没有好坏之分，网上有很多这样的引战帖，但成年人需要懂得明辨是非，客观辩证的看待问题。



## 博士项目
博士（Phd），是以学术研究为主要目的的学位，需要进行大量的科研，并进行理论上的创新，推动某一领域的进步。除欧陆外，大部分国家都欢迎本科生直接攻读博士（直博）。博士项目一般为全奖，如果对方院校不提供奖学金，也可以向国家留学基金委（[CSC](https://www.csc.edu.cn/)）申请国家公派留学奖学金，部分地区会给博士提供相当优渥的奖学金（如ETH,EPFL和HKPFS项目）。关于博士项目，有如下的特点

+ 对于基础学科的同学来说，博士是真正进入学科殿堂的起点，在基础学科领域想做出突破的成果非常困难，但这是推动人类社会发展的动力
+ 博士从来都不是逃避就业的港湾，相反，博士期间需要承受较大的心理压力，需要摸索学科知识的边界，如果不是强烈的兴趣支撑，请谨慎读博
+ 美国博士一般年限较长，有明确的论文要求，5-7年毕业，英联邦的硕士一般学制较短，没有明确的论文需求（是对于博士资格的侧重点不同，并不是水），如中国香港，新加坡的博士一般4-5年毕业，如果已经有硕士学位则是3年毕业；欧陆的博士一般需要先取得硕士学位
+ 申请博士需要在本科阶段就有良好的学术表现，近年来出国申请的难度越来越高，北美top20的博士基本需要很强大的学术背景或者推荐信，但这也不是说没有论文就不行，本科生直博的时候，大部分导师还是在考察一个人的学术潜力，所以一段完整的项目经历和一个良好的学业成绩是非常关键的。
+ 中国香港和新加坡为了拉拢中国大陆的生源，一般会在当年保送研究生之前进行一批提前批次的招生

## 关于语言成绩（IELTS, TOEFL, GRE）
语言成绩是刚入学的新生最关心的问题。这里主要就三个主流的考试IELTS, TOEFL, GRE进行阐述。

首先需要说明，语言成绩，过线即可，不要盲目追求过高的分数；另外，每个人的英语基础是不一样的，是否要报班的问题我们会在下面进行阐述。

+ IELTS：国际英语测试系统（International English Language Testing System），主要由[剑桥大学考试委员会](https://zh.wikipedia.org/wiki/%E5%89%91%E6%A1%A5%E5%A4%A7%E5%AD%A6%E8%80%83%E8%AF%95%E5%A7%94%E5%91%98%E4%BC%9A)<font style="color:rgb(32, 33, 34);">、</font>[英国文化协会](https://zh.wikipedia.org/wiki/%E8%8B%B1%E5%9C%8B%E6%96%87%E5%8C%96%E5%8D%94%E6%9C%83)负责。有效期为2年。分为四项考试：Listening，Reading， Writing和Speaking。备考资料主要为剑雅真题，后期也可以使用9分达人。雅思考试主要为纸笔考试，最近也推出了雅思机考模式，形式和纸笔考试基本类似。雅思的满分为9分，四个部分的满分也都为9分，最后的总分为平均数（逢0.25进位），一般来说6.5分以上是一个不错的成绩，很多院校也有小分的要求，比如小分不低于6。中国大陆的雅思报名通过[中国教育部雅思考试中心](https://ielts.neea.edu.cn/)进行。截至2021年8月，中国大陆雅思考试的报名费用为2170元
    - Listening：分为4个section，时间30分钟（听力）+10分钟（填答题卡），听力读一遍，主要题型有填空、匹配、选择、地图等。雅思的听力较为简单
    - Reading：3篇文章，40道题，考试时间60分钟，主要题型为填空、选择、匹配、判断等
    - Writing：2个部分，小作文为图表题、地图题或者工业流程题，时间20分钟，150词左右；大作文为论述题，主要就一个观点进行论证或阐述，40分钟，250词左右
    - Speaking：雅思口语考试为真人面对面对话。主要是三个部分，第一部分为日常对话，速问速答，第二部分为话题阐述，随机抽题，1分钟准备，2分钟表达，第三部分为思辨性讨论，主要就前两部分的内容进行一些讨论。雅思口语每个季度会换一次题库，每个季度内的口语题库是恒定的，每年的1月，5月，9月是换题季，尽量不要报考这个月份的考试



+ TOEFL：检定非英语为母语者的英语能力考试（Test of English as a Foreign Language），由由[美国教育测验服务社](https://zh.wikipedia.org/wiki/%E7%BE%8E%E5%9C%8B%E6%95%99%E8%82%B2%E6%B8%AC%E9%A9%97%E6%9C%8D%E5%8B%99%E7%A4%BE)（ETS）举办的英语能力测验。有效期为2年。分为四项考试和雅思类似。托福考试为上机考试（包括口语）。备考资料主要为TPO。满分为120分，每个部分30分，一般来说100+（口语23+）已经是一个相当有竞争力的分数了。中国大陆的托福报名通过[中国教育部托福考试中心](https://toefl.neea.edu.cn/)进行。截至2021年8月，中国大陆托福考试的报名费用为2100元。在2019年8月，托福进行了改革，将时长缩短。托福考试有加试环节，为听力或者阅读，也有一定几率出现双加试。加试的目的是方便调节每次考试的难度，使得分数更加客观。改革之后，托福考试结束时将提供参考的听力和阅读分数。
    - Listening：在没有加试的情况下，分为两个section，1短1长+1短2长，全部都是选择题（包含单选、双选和不定项），听力播放时看不到题目和选项，播放结束后听问题，选项出现在屏幕上进行选择。托福的听力难度较大，且托福考试非常看重学生的听力，在写作和口语中也会反复出现听力

---

    - Reading：没有加试的情况下3篇文章，30道题，考试时间54分钟，全部都是选择题
    - Writing：2个部分，综合写作为先阅读一篇文章，然后听一篇文章，根据听到的和刚才阅读到的文章进行归纳；独立写作和雅思大作文类似。
    - Speaking：分为一道独立口语和三道综合口语。独立口语是论述一个话题，有15s时间准备，说45s；综合口语与听力相结合，一篇是校园生活场景，两篇课堂学术场景话题，采用机器评分与人工评分结合的方式
+ GRE：美国研究生入学考试（<font style="color:rgb(38, 38, 38);">Graduate Record Examination），适用于申请理工科、人文社科、商科、法学等多个专业的硕士、博士以及MBA项目。有效期是5年。GRE不是语言考试，一般不可以替代语言，在申请北美，欧陆和新加坡的有关项目时需要提供GRE成绩。GRE分为三个部分：语文V、数学Q和写作A。其中语文和数学的满分为170分，最低分为130分，写作的满分为6分。一般而言，320+3.5是一个不错的GRE分数。GRE考试为自适应考试，考试顺序为A+VQVQV/QVQVQ，多出来的一次是加试。初始时，V和Q都是中等难度，如果考生在该部分完成良好，就会进入hard模式，也会相应得到更高的分数。主要复习资料为要你命3000单词以及考试机经</font>
    - verbal：分为两种题型，阅读和词汇填空，阅读20分钟10道题4篇文章，词汇10分钟10道题
    - quantative：数学较为简单，考前多刷题即可
    - Analysis：分为issue和argument，官方已经提供了题库

### 托福和雅思如何选择
托福和雅思的认可度都在越来越高，一般来说英联邦更认可雅思成绩，北美更认可托福成绩，但都不是绝对的。有几点因素可以考虑

+ 雅思的听力比较简单，如果听力基础本身不是很好的话，建议报名雅思考试
+ 雅思的小分比较难达到，比如剑桥大学要求小分7.0，由于中国大陆口语压分的缘故，口语7.0基本是相当流利的水平，这时候选择考托福就相对明智
+ 总体来说，托福的评分标准更加客观，分数也更为可信

### 我不出国，需要学习雅思托福嘛
这个视个人情况而定，不要被某些校代所迷惑。所谓的提升英语能力，帮助四六级考试本质上都make no sense。因为在备考过程中，大家都忙于应试刷题，真正提高英语的东西不多，且不说国内外英语考试的评分标准本就不同。但学习托福雅思，确实可以给自己留下后路，且是一种英语能力的证明，比六级考试更有说服力。有了语言成绩，如果想进行交换，也不会慌乱。

### 学习托福雅思GRE需要报班嘛
个人基础不同，想要达到的分数也不同。笔者也上过一些机构的班，也听过上过的同学的评价，可以供大家参考

+ 机构所谓的提升英语能力基本都是在扯淡，靠十几节课就能提升的英语能力的话，还要学十几年英语干什么。语言是需要感觉和长期训练的东西，报班只会教一些应试技巧
+ 机构有一定的监督作用，可以帮助同学快速上手考试，掌握考试规律和技巧
+ 不过，这些东西如果用心找的话，网上也都有现成的，直白的说，机构的知识壁垒并没有达到要付那个价钱的地步，前提是比较自律，有一定找信息的能力
+ 关于口语与写作：经费充足的话可以试试小班，或者万能的淘宝什么都有的，注意自己要甄别质量
+ 总而言之，报班与否取决于自己，问清楚自己是否具备这样的能力以后就能做出决定了



## 关于申请
申请季一般在保研以后，所以出国的同学是一定要放弃保研资格的

出国的材料准备一般是：CV(curriculum vitae) RL（reference letter）PS（personal statement）/ RS（research statement）

一般来说，授课硕士申请主要参考三维（GPA，语言，GRE）> 科研 = 实习 > 推荐信、connection

研究型硕士或博士申请主要是大牛推荐信、connection >> 科研（最好有产出） ≈ GPA > 语言

可以看到，无论是何种类型，GPA都占有较大的比重，所以刚入学的大一新生们一定要好好学习

### 留学中介 vs DIY
这个也是仁者见仁的问题，我提供一些思路：

+ 博士申请一般不需要留学中介，因为只需要和导师商量好就行
+ 好的留学中介会比较负责，早期负责软背景提升，中期负责文书撰写，帮忙进行网申，后期进行签证指导等一系列工作，这些工作都比较耗时，如果决定出国的时间较晚，且个人三维也还需要提升的话，可以选择一个靠谱的中介为自己分担
+ 不负责任的中介就是电子宠物，花钱图个安心的工具罢了
+ 如果自己搜集信息的能力较强，GPA语言科研等也都已经准备好了，DIY是没有任何问题的，可以找一些native speaker帮助自己修改润色文书
+ 北美的项目比较复杂，老师也比较多，如果觉得麻烦的话，中介也是一种选择
