# 前言
一作来自2024级工试，分流选择了自动化（当时自动化还在工管院），二作是光电拔尖本地人（乐）。基于两人对于光电专业和现代工学院的了解，写作此文章，介绍转光电（主要是针对大一）的方法和注意点。建议同时阅读《写给光电专业新生》。

# 一、二次拔尖
光电专业的二次拔尖是2024级刚有，第一届的数据大概是过初审40人，实验班接收30人，最终接收到32名（有2人放弃）。光电二拔的全名是：光电信息科学与工程（光电技术与信息材料实验班），为光电和材料物理主辅修制度（准出课均满足要求，毕业时可以自己选择一个专业作为主学位，另一个拿辅修学位；亦可以准出课修满其一，毕业直接选其准出，另一仅做知识储备）。光电二拔没有笔试，主要是看简历和面试。简历提交即初审方面，照着给的格式填写成绩与获奖信息等即可，无需过分担心。面试方面，并非所有过初审同学都要面试，初审通过的前60%同学无需面试；面试时注意自信，整个流程感觉像是和老师的交流谈话，不必紧张，大概谈一谈自己的选择本专业的原因及自我介绍即可，还会根据你前面简历内容进行简单提问，亦有与专业无关的像兴趣爱好之类的问题。另外，去年面试时会有有关专业常识的知识作答环节，10道题目选择一道进行简单分析回答，尽量做到逻辑清晰回答无纰漏，预计今年也会有相似的问题。

光电拔尖班的保研和光电普通班是分开计算的，目前因为是第一届，所以前面情况无从参考，只能从老师“画的饼”来讲。计划保研率是肯定会比普通班要高的，但是会根据我们班内学习情况来决定，有说能保70%的，也有说前50%就稳了的，甚至有在班会上提到“如果都学得好就都给保”这样的大饼的（可能性无限低）。总的来说比普通班高应该是板上钉钉的事，高多少则还有待观察。

学业安排上实验班也会有不同的待遇，譬如比大类更早学习电路分析等专业课程，大化不作为必修课而作为选修课，模电作为大二上必修放入课表，以后的物理课程安排据说也会进行调整（力热与光电磁各一个学期修完）。此外还有项目制课程贯穿三年，更早接触导师做项目，边学边努力做出成果，期望达到所学即所用。总的来讲课程压力会比普通班稍大，但能更早接触到专业知识。

光电拔尖班可以退出到大班进行二次转专业，流程与大一结束进行的正常转专业相同，可以理解为多了一次机会。报了拔尖，没能成功没有任何损失，成功了如果不想待在光电也可以继续转走，而且保研率和待遇还会多一层保障。

# 二、转专业
:::info
以下内容参考23级、24级跨大类准入的官方文件和24级光电培养方案。

:::

准入标准原文如下：

> 1. 须至少已修或者在修《普通物理（力学）》、19000200《大学化学》、《普通物理（热学）》三门课程中的任意一门，本学期（笔者tips：指大一下学期）结束时取得相应学分；
> 2. 未满足准入课程学分要求的，可根据申请者所修读数理化等理科基础课程的修读情况，决定是否给予面试资格。
>

注意到，准入标准1仅要求三选一，甚至没有提到通修微积分层次的问题，可能是他们写文件的时候疏忽了。但是微积分的这个“空子”确实是可以钻的，笔者的朋友亲身经历，微积分1学的二层次，微积分2学的一层次也转入成功了。不过基于光电对微积分的要求，还是建议大家直接学一层次，以免大二转入之后还要补。不过准入标准2表示可以不选准入课，但是笔者不建议这么做。笔者认为这太抽象了容易被卡人。

关于三节准入课，首先想从光电专业毕业，这三节课是全都要学的。但是普物力学和另外两节课相比比较特殊。大化和普物热学在仙林校区有开课，可以转入之后再学，普物力学只在大一上学期开课（鼓楼校区），转入之后还需要回鼓楼补课。所以笔者建议至少选上普物力学。光电的三个准入课都是有替代关系的，普物替代在此不表，可以自行查表；大化可以被化学学院的大学化学A+B替代。不过想转专业，准入课是不能被替代的，但是光电的准入课只要求三选一，这个时候我们也不太笃定能不能替代。也就是说，在大家转入光电之前**<u>最好不要</u>**去学这几个准入课的“上位课程”，不要赌可以替代。但是已经进入光电专业之后就可以去选上位课替代本专业的课（虽然一般没人这么做）。

光电转专业的流程和大多数专业一样，只有简历+面试。面试时间是在第二学期期末考试前几周，所以你第二学期的成绩老师是不知道的，可能会通过口头询问的方式来了解（比如问你线性代数的期中成绩之类）。

我们收集了好几位24级转光电成功的学姐/学长的数据（主要是面试），面试问的内容其实参考意义不太大，没有能总结出一些卡人的点。老师可能会问你各种稀奇古怪的问题，比如打不打游戏，大学物理实验课，甚至科学之光（都是亲身经历）；老师也可能会问你高中的情况，原专业的情况，来到新专业之后的打算等等。总之面试相当诡异。By the way，似乎现代工学院某一个老师不太喜欢打游戏的学生，所以如果问到这个问题建议坚决否认。

:::info
举个栗子，笔者朋友（已成功转入）微积分1学的二层次，微积分2学的一层次，微积分2期中还挂科了。老师问ta的高考成绩，上大学心情怎么样等等。笔者本人认为ta的面试过程就十分的抽象，这也能放进去，实在不知道24级为什么13进8卡掉了5个人。

:::

光电专业也可以大一下学期再转，因为这个学期有且仅有普物热学一门准入课。无论什么时候打算转光电，笔者都建议把通修数学（微积分一层次，线性代数一层次）成绩卷高一点，这肯定能让你在现工的传奇面试中获得较为领先的地位。
