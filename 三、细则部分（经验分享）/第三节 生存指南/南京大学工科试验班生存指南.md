# 前言
笔者是南京大学23级工科试验班学生，现已分流进入现工院光电信息科学与工程专业学习，现将笔者这一年来在工试大类学习的心得整理如下，供各位参考，本文仅代表个人观点，如有谬误之处，请各位批评指正。

# 一、工科试验班是个怎样的大类
工科试验班分流方向有三种：现代科学与工程学院（简称现工院）、工程管理学院（简称工管院）、建筑与城市规划学院（简称建规）。现工院包括光电信息科学与工程、新能源科学与工程、材料物理、材料化学、生物医学工程五个专业，工管院包括金融工程、自动化、工业工程三个专业，建规包括建筑学和城乡规划两个专业。这十个专业构成了工科试验班这一大类，其中，建规是极少数的分流方向，笔者对此并不了解，在此篇指南中不加以讨论。可以看出，工科试验班并不是“纯”的工科，单单工程管理学院就会授予工学、管理学、经济学三种学位。

# 二、工科试验班的分流情况
23级工科试验班二拔后266人，其中进入材料化学6人，材料物理29人，光电信息科学与工程50人，生物医学工程13人，新能源科学与工程30人，工业工程29人，自动化29人，金融工程20人，城乡规划3人，建筑学0人，转专业47人。其中光电、自动化、新能源、工工报满。光电、新能源第一志愿报满，工业工程疑似报满（笔者的朋友在工工进行了调查，样本均为一志愿，但笔者未曾听说一志愿被工工刷掉的情况）。光电与自动化准入rank均在百分之70左右（仅代表23年）。可以看出，光电与自动化在工试里属于比较热门的专业，同时工试保持大类里较高的支援满足率。笔者认为是专业课较少的大一导致同学们转专业的意向与成功率均较高，从而使得名额相较于大类内分流的名额较为充足。

# 三、工科试验班的课程设置
笔者可以在这里断言，工科试验班的课程设置是理工科中最利于转专业的，没有之一。工试自带一二层次微积分，同时没有除通修课外的自选课程，如果一心想转专业，可以一门准入课都不选。分流进入现工院，需要修读普物力学、普物热学、大学化学三门中的一门。分流进入自动化，需要修读自动化导论。分流进入工业工程与金融工程，需要修读数据分析决策导论。其中，只有普物力学，大学化学是在上学期修读。所以，即使你第一学期发现自己转专业不达成要求，或者不想转专业，在第二学期选上准入课，也可以进入工管和现工的任意一个专业进行学习。

此外，工试大一上的通修课程包括一层次微积分、通修c、英语、思政，这也是工试分流的成绩依据（仅看上学期），大一下包括线代、马原、微积分、英语。其中，笔者特别强调c语言选好老师的重要性，两位老师给分天差地别。很遗憾，笔者上的是给分差的那一位的课（笑）。对于除c语言外，笔者不加赘述。

对于平台课，笔者对现工院的较为熟悉。很遗憾，这三门课质量并不高，上课主要以读ppt为主。至于课程难度，现工院的物理难度虽远低于物院，但是高于学习普通物理、大学物理这两门课的学院（匡院传奇课程大学物理除外）如果高中时就很害怕物理的同学，笔者不建议你进入现工院学习。

至于自动化导论和数据分析与决策导论，两门课的硬核程度与现工的三门差了很多。自动化导论没有期末考试，数据分析与决策导论笔者并不了解，但是据工管朋友所说，也是一门大水课。

# 四、学习策略
笔者认为，工试的同学们在大一最应该做的事，就是搞清楚自己喜欢什么，从而确定自己是转专业，或是分流进入某个学院学习某个专业。工试每个方向差别较大，希望同学们能搜集资料，以及询问学长学姐，得到更多信息。

相比别的专业，大一的工试确实很轻松，但相比工试别的年级，工试的大一也并不轻松。大一上的微积分1，C语言（无基础的情况）拿高分都是需要付出不少努力的。大一下的微积分2，线性代数，想得到90+的分数，难度还是不小的。若想进入现工院学习，大一下最好修读“传奇课程”大学物理实验，即使他既耗时又学不到东西。依笔者之见，大一的空闲对专业选择是好事，但是对于升学并不是好事，大二甚至大三才接触专业核心课程，会导致同学们的科研与竞赛经历一片空白，这在外保和出国中会处于极大的劣势。因此，我建议同学们在空余时间，对你想分流进入的那个专业，提前学习核心课程，不要求达到能考试拿高分的水平，只是利于你大二大三进组科研。当然，若你只想保研本校，甚至不想保研，笔者这番话可以自行忽略。

笔者在这里强调微积分与线代的重要性。14个学分的课程，对于你大一的gpa，无疑会留下浓墨重彩的一笔。对于以后的学习与科研，这两项用的最多的数学工具也至关重要。

# 五、笔者寄语
大家读到这里，也许会发现，笔者并没有谈就业之类的问题，这也许恰恰是某些同学最关心的话题。笔者的观点里，专业的就业是放在第二位的，你自己的兴趣才是要放在第一位的。笔者反对为了转专业而转专业的行为，转专业一定要慎重。对于工试大部分专业，都不是张雪峰等人口中最热门的那一批，但是都可以说的上尚可。因此，如果没有特别想去的专业，笔者不建议工试的同学们盲目的转出。“条条大路通罗马”，笔者祝愿工试的每位同学能找到自己的专业兴趣所在，能进入理想的专业学习。

一些想说的话大概就说到这里，希望能对大家有所帮助，有其他感到疑惑的地方，欢迎来向我提问。衷心祝大家在工试大类不虚此行，奔向美好的未来。

# 六、工试转专业补充
我愿称工科试验班为南大最方便转专业的地方，请来到工试的同学们不要灰心。首先，工试没有专业必修课。由于工试的三个分流院系所需准入课几乎没有重合，所以上下两学期的所有准入课均为“可选”，因此工试的同学们能够all in转专业，直接将自己当作目标院系的同学。其次，在工试，转专业失败的后果并不太严重，你并不会因为失败而被发配到特别差的专业；而且，由于工试大一的课业较少，转专业失败后进入对应院系并不需要补很多课程；不仅如此，这几个院系善良的教务员甚至可能愿意支持你大二继续all in转专业(当然，失败风险自担)。如此看来，工试为南大元培实至名归。
