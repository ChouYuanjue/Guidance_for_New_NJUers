# <font style="color:rgba(0, 0, 0, 0.9);">行</font>
## 校内出行
浦口校区其实并不算是很大，大家平时主要活动区域也就是宿舍区-教学区两点一线，根据我的个人体验，步行足矣。另外，校园内的共享电动车、共享单车等资源都很充足，所以我觉得并不需要买电动车或者自行车。

## 校外出行
南1门和南2门门口都有公交车，能很方便地前往周边商区或者地铁站。

单纯从距离上来说，校园离S8的“高新开发区”站更近，骑车大概不到10分钟的样子。

但是3号线进城换乘更方便，所以公交或者骑车到3号线“东大成贤学院”站也是不错的选择。

但是！晚上回学校的时候，一定要注意末班车的停运时间！

其余请参考[《浦口校区交通指南》](https://www.yuque.com/greatnju/q-a/lbim7zsai7t0c8ln?singleDoc# 《浦口校区交通指南》)



# 学
## 教学楼概况
上课地点主要在**教学楼**和**玉辉楼**，具体位置如图：

![](images/1faf2303155246a0a26b270a3feb9ccd.png)

课表上的“浦1”“浦2”就是教学楼的意思，1和2指的是分区。

![](images/d8d060cb522a48da8cd12cf867ebe03a.png)

虽然根据地图显示浦1、浦2、浦3相互连通，理论上从任意一个门进教学楼，绕啊绕啊绕总能绕到自己要去的教室，但是因为浦3目前不开放，很多门都是锁着的，为了避免绕到“此路不通”，还是建议就近找门进教学楼比较方便。教室在浦1就从浦1进，教室在浦2就从浦2进。

教学楼内有自助打印机，可以方便大家打印各类证明文件、成绩单等等，具体位置大概在大平台，靠近教室浦1-230；

教学楼内有饮水机，免费使用，无需刷卡，冷热水均有；

教学楼内有自动贩卖机，主要售卖的内容是各类饮料，但是分部较为稀疏。

根据教学楼内的标识，教学楼开放时间大概在7:00-22：30；

浦口教学楼内有电梯。

# 购
## 附近大型商圈
附近较为大型的商圈有龙湖天街、弘阳广场等。

## 网购
网购的话，快递地址是：**江苏省南京市浦口区学府路8号南京大学浦口校区。**

快递会送到第一菜鸟驿站/第二菜鸟驿站。两个菜鸟驿站其实位置都在一起，在食堂的地下车库里；

如果你收到的取件码是**1-n-xxxx**，那就是第一菜鸟驿站的意思；

如果你收到的取件码是**n（n≠1且n≠7且n≠8）-a-xxxx**，那就是第二菜鸟驿站的意思；

如果你收到的取件码是**n-a-xxxx（n=7或者n=8）**，意味着你的快递是邮政快递，需要前往浦口教学楼1楼领取，具体位置大约如图：

![](images/fb2e49ae84f74c4bb31ca6513b5a16fd.png)

![](images/a3b80856e37d4e1f8abe68d221eff1c5.png)

**京东与天猫超市**的快递一般会送到第一菜鸟驿站；

网购退货也是去菜鸟驿站退，当然也可以选上门取件，一般会到宿舍门口来拿；
